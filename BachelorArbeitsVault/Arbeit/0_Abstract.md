# Bachelor-Thesis
Entwicklung und Untersuchungen zur Nutzbarkeit eines Rollstuhls als Eingabegerät zur Navigation im virtuellen Raum.


## Languages
- C/C++
- C#

## Software/Frameworks
- Visual Studio Code
- PlatformIO
- Blender
- Arduino-Framework
- .Net 6.0
- Obsidian
